## üì¶ Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/vinaypatle/portfolio.git
cd portfolio
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
